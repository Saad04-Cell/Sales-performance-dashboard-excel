# Interactive Sales Performance Dashboard

## Project Overview

This project is a Reusable Sales Performance Dashboard built in Microsoft Excel using Power Query, Data Model, PivotTables, Pivot Charts, and Slicers.

The goal of this project was not just to analyze data, but to build a reporting system that can:

* Accept new data anytime
* Refresh automatically
* Scale to different business scenarios
* Support data-driven decision-making

The dashboard transforms raw sales transaction data into actionable business insights through interactive visualizations and KPI tracking.

## Dashboard Features

### KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* Total Customers

### Interactive Analysis

* Monthly Sales Trend
* Sales by Region
* Profit by Category
* Customer Segment Analysis
* Top 5 Products
* Top 5 Customers

### Dashboard Functionality

* Interactive slicers for Region, Category, and Segment
* Automatic dashboard updates after data refresh
* Reusable reporting structure
* Business-friendly visualizations

---

## Key Business Insights

* The dashboard identifies the highest-performing sales regions.
* Customer segments can be analyzed to understand revenue contribution.
* Top customers and products are highlighted for business focus.
* Profitability can be monitored across product categories.
* Sales trends can be tracked over time for performance evaluation.
